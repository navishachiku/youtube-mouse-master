# YouTube Mouse Master

YouTube 플레이어 사용자를 위한 상호작용 강화 스크립트입니다. 고성능, 제로 오버레이(Zero-Overlay) 및 고도의 커스텀 기능을 지원하는 Tampermonkey 스크립트입니다.

## ✨ 주요 기능

* **빠른 제어**: 클릭 및 휠 스크롤과 같은 마우스 동작에 대응하는 커스텀 액션 존을 플레이어에 설정하여 볼륨, 속도, 재생 위치 등을 빠르게 조정할 수 있습니다.

* **커스텀 액션 존**: 감지 영역 설정을 자유롭게 조정할 수 있습니다. 영역 크기와 위치를 자유롭게 변경할 수 있습니다 (기본값으로 왼쪽, 중앙, 오른쪽 3개 영역 설정 제공).

* **제로 오버레이 상호작용 (Zero-Overlay)**: 기존의 투명 레이어 덮어쓰기 방식을 버리고 고성능 좌표 계산 방식을 채택하여 프로그레스 바나 버튼 등 기본 UI 클릭을 방해하지 않습니다.

* **Mac / 고주파 휠 최적화**: 필터링 메커니즘이 내장되어 MOS, SmoothScroll 또는 Mac 트랙패드에 완벽하게 대응하며 과도한 민감도로 인한 튕김 현상을 방지합니다.

## ⚙️ 설정 가능한 파라미터

스크립트 상단의 `SETTINGS` 및 `CONFIG` 블록에서 직접 설정을 변경할 수 있습니다.

### 글로벌 설정 (Global Settings)

| 파라미터 | 설명 | 기본값 |
| :--- | :--- | :--- |
| `DEBUG` | 콘솔에 디버그 메시지 출력 여부 | `false` |
| `ZONE_TOGGLE_KEY` | 영역 시각화 활성화/비활성화 단축키 | `F9` |
| `OSD_DURATION` | OSD 알림이 화면에 머무는 시간 (ms) | `800` |
| `OSD_FADE_OUT` | OSD 서서히 사라지는 애니메이션 시간 (ms) | `150` |
| `OSD_FONT_SIZE` | OSD 텍스트 글꼴 크기 (px, em, rem 등 지원) | `28px` |
| `USE_WHEEL_COUNT_FIXED` | 고정 휠 카운트 필터링 사용 여부 (Mac 사용자 권장) | `false` |
| `WHEEL_DELAY` | 휠 이벤트 디바운스 지연 시간 (ms) | `1` |
| `WHEEL_COUNT_THRESHOLD` | 휠 카운트 트리거 임계값: 동작 실행 전 누적할 휠 이벤트 수 | `14` |

### 커스텀 영역 설정 (Custom Zone Configuration)

개인적인 필요에 따라 액션 영역의 크기와 위치를 완전히 커스텀할 수 있습니다.

기본적으로 왼쪽, 중앙, 오른쪽 3개 영역 설정을 제공합니다:

| 영역 | 왼쪽 클릭 | 오른쪽 클릭 | 휠 동작 |
| ----- | ----- | ----- | ----- |
| **왼쪽 (볼륨)** | 최대 볼륨 (100%) | 빠른 음소거 (0%) | 볼륨 조정 +/- 5% |
| **중앙 (재생 위치)** | 기본 동작 (재생/일시정지) | 기본 동작 (콘텍스트 메뉴) | 재생 위치 이동 +/- 5초 |
| **오른쪽 (재생 속도)** | 빠른 2.0x | 초기화 1.0x | 재생 속도 조정 +/- 0.25x |

### 지원 액션 목록 (Supported Actions List)

`mouse_action` 내에서 사용할 수 있는 `action` 종류는 다음과 같습니다:

| 액션 이름 (action) | 설명 | 파라미터 예시 (value) |
| :--- | :--- | :--- |
| `volume_up` | 볼륨 증가 | `5` (+5% 의미) |
| `volume_down` | 볼륨 감소 | `5` (-5% 의미) |
| `volume_set` | 고정 볼륨 설정 | `0` (음소거) 또는 `100` (최대) |
| `volume_mute` | 음소거 전환 / 해제 | 파라미터 필요 없음 |
| `seek` | 재생 위치 이동 | `5` (앞으로) 또는 `-5` (뒤로) |
| `toggle_play_pause` | 재생 / 일시정지 상태 전환 | 파라미터 필요 없음 |
| `speed_up` | 재생 속도 증가 | `0.25` |
| `speed_down` | 재생 속도 감소 | `0.25` |
| `speed_set` | 고정 재생 속도 설정 | `1.0`, `2.0` 등 |
| `none` | 아무 작업도 하지 않음 | 이벤트를 YouTube 기본 처리 방식으로 넘깁니다 |

## 📦 설치 방법

**방법 1: 원클릭 설치 (권장)**

1. 브라우저 확장 프로그램 [Tampermonkey](https://www.tampermonkey.net/)를 설치합니다.
2. **[GreasyFork 스크립트 페이지](https://greasyfork.org/en/scripts/566499-youtube-mouse-master)**를 방문합니다.
3. **"이 스크립트 설치"** 버튼을 클릭합니다.

**방법 2: 수동 설치**

1. Tampermonkey에서 "새 스크립트 만들기"를 클릭합니다.
2. `YouTubeMouseMaster.user.js`의 내용을 복사하여 붙여넣습니다.
3. 저장하면 완료됩니다!
