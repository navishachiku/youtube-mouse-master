# YouTube Mouse Master

パワーユーザー向けに設計された YouTube プレイヤー操作強化スクリプト。高度に最適化され、オーバーレイなし（Zero-Overlay）かつ自由にカスタマイズ可能な Tampermonkey スクリプトです。

## ✨ 主な特徴

* **クイックコントロール**: プレイヤー上にカスタムアクションゾーンを設定し、クリックやホイール操作で音量、速度、シークなどを素早く調整できます。

* **カスタムアクションゾーン**: センサーゾーンの設定を高度にカスタマイズ可能。サイズや位置を自由に変更できます（デフォルトでは左・中・右の3ゾーン設定を用意）。

* **ゼロオーバーレイ操作**: 従来の透明レイヤーでのカバーを廃止し、高性能な座標計算を採用。プログレスバーやボタンなどのネイティブ UI のクリックを一切妨げません。

* **Mac / 高頻度ホイール最適化**: フィルタリング機能を内蔵。MOS、SmoothScroll、または Mac のトラックパッドに最適化されており、過敏な動作を防ぎます。

## ⚙️ カスタマイズ可能なパラメータ

スクリプトップの `SETTINGS` および `CONFIG` ブロックで直接設定を変更できます。

### グローバル設定 (Global Settings)

| パラメータ | 説明 | デフォルト値 |
| :--- | :--- | :--- |
| `DEBUG` | コンソールにデバッグメッセージを出力するか | `false` |
| `ZONE_TOGGLE_KEY` | ゾーンの可視化を切り替えるホットキー | `F9` |
| `OSD_DURATION` | OSD 表示が画面に留まる時間 (ms) | `800` |
| `OSD_FADE_OUT` | OSD のフェードアウトアニメーション時間 (ms) | `150` |
| `OSD_FONT_SIZE` | OSD テキストのフォントサイズ (px, em, rem などに対応) | `28px` |
| `USE_WHEEL_COUNT_FIXED` | 固定ホイールカウントフィルタリングを有効にするか (Mac ユーザー推奨) | `false` |
| `WHEEL_DELAY` | ホイールイベントのデバウンス遅延時間 (ms) | `1` |
| `WHEEL_COUNT_THRESHOLD` | ホイールカウントの実行しきい値：累積したイベント数でアクションを実行 | `14` |

### カスタムゾーン設定 (Custom Zone Configuration)

個人のニーズに合わせて、アクションゾーンのサイズや位置を完全にカスタマイズできます。

デフォルトでは以下の左・中・右の3ゾーンが設定されています：

| ゾーン | 左クリック | 右クリック | ホイール操作 |
| ----- | ----- | ----- | ----- |
| **左側 (音量)** | 最大音量 (100%) | クイックミュート (0%) | 音量ステップ +/- 5% |
| **中央 (シーク)** | パススルー (ネイティブの再生/一時停止) | パススルー (ネイティブのメニュー) | シーク +/- 5秒 |
| **右側 (再生速度)** | クイック 2.0x | リセット 1.0x | 速度ステップ +/- 0.25x |

### サポートされているアクション一覧 (Supported Actions List)

`mouse_action` 内で使用可能な `action` タイプは以下の通りです：

| アクション名 (action) | 説明 | パラメータ例 (value) |
| :--- | :--- | :--- |
| `volume_up` | 音量を上げる | `5` (+5% を意味します) |
| `volume_down` | 音量を下げる | `5` (-5% を意味します) |
| `volume_set` | 固定の音量を設定 | `0` (ミュート) または `100` (最大) |
| `volume_mute` | ミュート / ミュート解除の切り替え | パラメータ不要 |
| `seek` | シーク移動 | `5` (進む) または `-5` (戻る) |
| `toggle_play_pause` | 再生 / 一時停止の切り替え | パラメータ不要 |
| `speed_up` | 再生速度を上げる | `0.25` |
| `speed_down` | 再生速度を下げる | `0.25` |
| `speed_set` | 固定の再生速度を設定 | `1.0`, `2.0` など |
| `none` | 何もしない | YouTube ネイティブの動作を優先します |

## 📦 インストール方法

**方法1：ワンクリックインストール（推奨）**

1. ブラウザ拡張機能 [Tampermonkey](https://www.tampermonkey.net/) をインストールします。
2. **[GreasyFork スクリプトページ](https://greasyfork.org/en/scripts/566499-youtube-mouse-master)** にアクセスします。
3. **「このスクリプトをインストール」** ボタンをクリックします。

**方法2：手動インストール**

1. Tampermonkey で「新規スクリプトを作成」をクリックします。
2. `YouTubeMouseMaster.user.js` の内容をコピーして貼り付けます。
3. 保存すれば完了です！
